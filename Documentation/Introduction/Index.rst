Introduction
============

This documentation was written for version 1.0.0 of the extension.


What does it do?
----------------


Creates Geckoboard widgets with data from TYPO3:
~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~~

This extension pushes TYPO3 data to Geckoboard. Statistics such as the count of active pages are included
and also informative displays such as a sliding text element of the last 10 updated pages.
